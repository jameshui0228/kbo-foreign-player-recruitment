# Candidate Shortlist Schema

후보군 shortlist는 포지션별로 최소 다음 컬럼을 포함한다.

| column | description |
|---|---|
| rank | role-specific rank |
| player_name | player name |
| age | age on evaluation date |
| position_role | hitter position or pitching role |
| current_team | current club |
| league | MLB/MiLB/NPB/Farm/CPBL/ABL/etc |
| bats_throws | bats/throws or throws |
| key_strength | strongest baseball skill |
| acceptable_flaw | flaw that may make acquisition realistic |
| ssg_fit_score | 0-100 |
| kbo_translation_score | 0-100 |
| skill_score | 0-100 |
| availability_score | 0-100 |
| risk_score | 0-100, higher means safer |
| final_score | 0-100 |
| short_comment | one-sentence interpretation |

## Required Role Outputs

- Foreign hitter Top 5-10
- Foreign starter Top 5-10
- Asian quota reliever Top 5-10
- Final 1st/2nd/3rd priority per role
